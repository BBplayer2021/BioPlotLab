# Node.js 和 npm 安装指南（Windows）

## 方法一：从官网下载安装（推荐）

### 步骤：

1. **访问 Node.js 官网**
   - 打开浏览器，访问：https://nodejs.org/
   - 或直接访问中文站：https://nodejs.org/zh-cn/

2. **下载安装包**
   - 点击下载 **LTS 版本**（长期支持版，推荐）
   - 选择 Windows Installer (.msi) 64-bit

3. **运行安装程序**
   - 双击下载的 `.msi` 文件
   - 按照安装向导提示操作
   - **重要**：确保勾选 "Add to PATH" 选项（通常默认已勾选）

4. **验证安装**
   - 安装完成后，重新打开 PowerShell 或命令提示符
   - 运行以下命令验证：
   ```powershell
   node --version
   npm --version
   ```
   - 如果显示版本号，说明安装成功！

---

## 方法二：使用 winget（Windows 包管理器）

如果您使用的是 Windows 10/11，可以使用内置的 winget：

```powershell
winget install OpenJS.NodeJS.LTS
```

安装完成后，重新打开终端并验证。

---

## 方法三：使用 Chocolatey（如果已安装）

如果您已经安装了 Chocolatey 包管理器：

```powershell
choco install nodejs-lts
```

---

## 安装完成后

1. **重新打开终端**（重要！）
   - 关闭当前的 PowerShell 窗口
   - 重新打开一个新的 PowerShell 窗口

2. **验证安装**
   ```powershell
   node --version
   npm --version
   ```

3. **进入项目目录并安装依赖**
   ```powershell
   cd C:\Users\Administrator\Desktop\project\16.landing_page
   npm install
   ```

4. **启动开发服务器**
   ```powershell
   npm run dev
   ```

## 常见问题

### Q: 安装后仍然提示找不到命令？
**A:** 请确保：
- 已重新打开终端窗口（安装后需要重启终端）
- 安装时勾选了 "Add to PATH" 选项
- 检查环境变量 PATH 中是否包含 Node.js 的安装路径

### Q: 如何检查 PATH 环境变量？
**A:** 
1. 右键"此电脑" → "属性" → "高级系统设置"
2. 点击"环境变量"
3. 在"系统变量"中找到 `Path`，查看是否包含 Node.js 路径（通常是 `C:\Program Files\nodejs\`）

### Q: 安装哪个版本？
**A:** 推荐安装 **LTS（长期支持）版本**，目前最新的是 v20.x 或 v18.x 版本。
