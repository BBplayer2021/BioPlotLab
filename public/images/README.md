# 图片文件说明

请将以下图片文件放置在此目录中：

## 必需文件

1. **volcano-default.png**
   - 描述：原生代码输出的火山图（带网格、默认配色）
   - 尺寸建议：1200x900px 或更高分辨率
   - 格式：PNG（支持透明背景）

2. **volcano-nature.png**
   - 描述：BioPlotLab 复现的 Nature 风格精美火山图
   - 尺寸建议：1200x900px 或更高分辨率（与 volcano-default.png 相同尺寸）
   - 格式：PNG（支持透明背景）
   - 要求：视觉上应该与 volcano-default.png 形成明显对比，展示从"业余"到"顶级期刊"的转变

## 图片要求

- 两个图片应该具有相同的尺寸和比例（建议 4:3）
- 确保图片清晰，适合在高分辨率屏幕上显示
- 建议使用 PNG 格式以保持图片质量

## 注意事项

如果图片文件不存在，组件会显示占位符提示信息。
